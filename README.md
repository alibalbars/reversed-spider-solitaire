## Installation
- Clone project
```sh
git clone https://github.com/alibalbars/reversed-spider-solitaire-ali-balbars.git
```

``` bash
# install dependencies
npm install

# start with hot reload at localhost:3000
npm start

# build for production with minification
npm run build
```
